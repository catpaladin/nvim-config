# dotfile setup

Setup in the config directory ($HOME/.config/nvim)

## Setup Fonts
```bash
./scripts/install_font.sh
```

## Setup Tmux & Alacritty
```bash
./scripts/install.sh
```

## Setup Go tools
```bash
./scripts/install_go_deps.sh
```

## Setup Python dependencies
```bash
pip install -r requirements.txt
```
